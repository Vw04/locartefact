# Geolore Brand Guidelines

## Logo

The Geolore logo consists of an `earth-outline` globe icon (Ionicons) paired with the wordmark "Geolore" in a rounded, bold sans-serif. The globe and text are rendered horizontally aligned.

### Sizes
| Variant | Icon Size | Font Size | Font Weight | Usage |
|---------|-----------|-----------|-------------|-------|
| `lg` | 48px | 40px | 800 | WelcomeScreen hero |
| `md` | 32px | 28px | 700 | Settings header |
| `sm` | 22px | 22px | 700 | Main app header |

---

## Primary Palette — Dark (main theme)

| Token | Hex | Usage |
|-------|-----|-------|
| Background | `#0D2218` | All screen backgrounds |
| Card / Input | `#162E20` | Cards, text inputs, elevated surfaces |
| Button | `#1A3828` | Primary CTA buttons |
| Border (inactive) | `#1E5038` | Chip borders, input borders at rest |
| Accent / Teal | `#2A9D8F` | Active borders, links, slider track, active chips |
| Text | `#FFFFF0` | All primary text (warm ivory) |
| Text Dim | `rgba(255,255,240,0.55)` | Subtitles, hints, disabled states |
| Error | `#ff7777` | Validation errors |
| Dev bar | `#050F0A` | Dev-only city selector bar |

---

## Typography

- **Font family**: System Helvetica (`fontFamily: 'Helvetica'`)
- **Primary text**: 14–16px, `#FFFFF0`
- **Titles / headers**: 28–52px, weight 700–800
- **Taglines**: italic, 14px, `#FFFFF0`
- **Dim / secondary**: `rgba(255,255,240,0.55)`
