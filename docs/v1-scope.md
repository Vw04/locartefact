# V1 Scope

## Included
- Personal use only
- iPhone first
- React Native + Expo (TypeScript)
- Local install from Xcode (free Apple Account)
- Supabase free tier for persistence
- GitHub for docs, config, and source control
- Manual tracking toggle
- Nearby named places + broader geographic context
- Manual refresh supported
- Notifications supported (after manual refresh works)
- Up to 5 facts per stationary session at ~1-minute intervals
- Feed-first experience

## Excluded from V1
- App Store / TestFlight distribution
- Map view
- Restaurant / business discovery mode
- Audio narration
- Offline caching
- Multi-user support
